# Bracket Styles

A project to see which style of opening bracket placement is more popular in the `R` development community, this:

    function(...) {
        ...
        return()
    }
    
    if ... {
        ...
    } else {
        ...
    }
    
    while ... {
        ...
    }

or this:

    function(...)
    {
        ...
        return()
    }
    
    if ...
    {
        ...
    }
    else
    {
        ...
    }
    
    while ...
    {
        ...
    }

Visit [here](https://kiendang.github.io/bracket-style-war/).
